---

## 🚀 Live Demo
🔗 [View Portfolio](https://ayushkothariportfolio.vercel.app/)

## 📌 Features
- Responsive design (mobile + desktop)
- Fast performance with Vite
- Clean and modern UI
- Projects showcase section
- Contact form / social links
